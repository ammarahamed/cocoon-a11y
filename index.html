Why Cocoon A11y exists

Some accessibility tools charge $49/month or more, and a few are entirely region-blocked. Others cost hundreds per year. Meanwhile, millions of websites have zero accessibility features.

Accessibility is a right, not a premium feature. Cocoon A11y is free for non-commercial use and works on any website with a single script tag. No accounts, no tracking, no cookies, no external requests.

How it works

  • Add one script tag to your HTML
  • The widget appears in the corner of your page
  • Users click it to access 25+ accessibility features
  • Everything runs client-side — no data leaves the browser
  • Fully configurable via data attributes

Configuration examples

Position the widget in the bottom-left corner with a blue accent:

<script src="a11y.js" data-position="bottom-left" data-color="#3b82f6"></script>

Persist user preferences across page loads:

<script src="a11y.js" data-persist="true"></script>

Browser Support

Cocoon A11y works on all modern browsers:

  • Chrome 80+ (including chunked speech keep-alive fix)
  • Firefox 78+
  • Safari 14+
  • Edge 80+
  • Mobile browsers (responsive panel)

Features that require specific APIs (like Web Speech) gracefully degrade — the section simply hides if the browser doesn't support it.
